## Bank Loan Analysis
Conducted an end-to-end analysis of bank loan data to identify factors influencing loan repayment and default risk. Used Python, Pandas, and Seaborn to perform exploratory data analysis and build visual insights on borrower income, employment stability, loan purpose, and debt-to-income ratios. The analysis helps financial institutions make data-driven lending decisions and reduce credit risk.

## 📁 Files

- `bankanalysis.ipynb`: The main analysis notebook

## 📊 Project Overview

- Analyzed historical bank loan data to understand borrower behavior and repayment patterns.
- Explored key borrower attributes such as income, employment history, loan purpose, interest rate, and debt-to-income ratio (DTI).
- Identified factors that influence loan repayment and loan default risk.
- Performed exploratory data analysis (EDA) to uncover patterns and relationships in loan data.
- Generated insights that help financial institutions assess borrower risk more effectively.
- Provided data-driven findings to support better loan approval and credit risk decisions.
- Helped identify high-risk borrower profiles that may lead to loan defaults.
- The analysis contributes to improving loan portfolio performance and reducing financial risk for banks.

## 🖼️ Sample Visualizations

- Loan Status Distribution – Bar chart showing fully paid vs defaulted loans.
